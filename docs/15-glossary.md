# Glossary

- xG: expected goals
- xA: expected assists
- EV0: fair odds derived from model probability, 1 / p
- Vig, overround: bookmaker margin embedded in prices
- CLV: closing line value, how your taken price compares to a closing price
- Snapshot: a data record keyed by as_of_utc, representing what was known at that time
- Leakage: using information not available at decision time
- Scenario mixture: pricing across start and bench cases using probabilities
